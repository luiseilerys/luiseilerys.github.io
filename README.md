# Luis & Eilerys · Nuestro Camino

Sitio web conmemorativo para Luis Angel Jimenez Escalona y Eilerys Adriana Reyna Meriño, mostrando su historia, perfiles, galería de fotos y un libro de visitas interactivo.

## ✨ Características
- **Diseño responsive** con Tailwind CSS
- **Pestañas** con información personal, juntos y galería
- **Galería de imágenes** con efecto hover
- **Libro de visitas** con almacenamiento local (los mensajes se guardan en el navegador)

## 🚀 Cómo usar
1. Descarga todos los archivos manteniendo la estructura de carpetas.
2. Abre `index.html` en cualquier navegador moderno.
3. Para publicar en GitHub Pages, sube la carpeta completa a un repositorio y activa GitHub Pages desde la configuración.
